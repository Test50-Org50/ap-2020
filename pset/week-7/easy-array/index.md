# C. Easy Array

## Problem Setup

- login to [CS50 IDE](https://cs50.dev)
- write `cd` in the *terminal* to go to the home directory
- write `mkdir easy-array` to create a folder called `easy-array`
- write `cd easy-array` to go to the `easy-array` folder
- write `code easy-array.cpp` to create a file called `easy-array.cpp` and open it in the editor

## Problem

Given two numbers \\(N\\) and \\(M\\), a \\(2D\\) array of size \\(N * M\\).

## Input

First line contains two numbers \\(N, M (2 \\leq N, M \\leq 100)\\) \\(N\\) donates number of rows and \\(M\\) donates number of columns.

Each of the next \\(N\\) lines will contain \\(M\\) numbers \\((1 \\leq A_i \\leq 10^3)\\).

## Output

print the sum of the elements of this array.

## Example

**Input:**

```text
3 3
1 2 3
4 5 6
7 8 9
```

**Output:**

```text
45
```
